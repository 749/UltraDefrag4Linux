HMENU WgxBuildMenu ( WGX_MENU *  menu_table,
HBITMAP  bitmap 
)

Builds menu from user defined tables.

Parameters
[in]menu_tablepointer to array of WGX_MENU structures. All fields of the structure equal to zero indicate the end of the table.
[in]bitmaphandle to bitmap intended to extract menu icons from.
Returns
Handle of the created menu, NULL indicates failure.
Note
The following flags are supported:
  • MF_STRING - data field must point to zero terminated Unicode string.
  • MF_SEPARATOR - all fields are ignored.
  • MF_POPUP - id field must point to another menu table describing a submenu.
