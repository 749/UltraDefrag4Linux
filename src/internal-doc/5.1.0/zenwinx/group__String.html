char winx_toupper ( char  c)

Reliable _toupper analog.

MSDN states: "In order for toupper to give the expected results, __isascii and islower must both return nonzero". winx_toupper has no such limitation.

Note
Converts ASCII characters only (as well as _toupper function included in ntdll library).
