void winx_path_remove_extension ( char *  path)

Removes the extension from a path.

If path contains as input \??\C:\Windows\Test.txt, then path contains as output \??\C:\Windows\Test. If the file name contains no dot or is starting with a dot, then path keeps unchanged.

Parameters
[in,out]paththe native ANSI path to be processed.

Removes the file name from a path.
