int winx_init_library ( void *  peb)

Initializes zenwinx library.

If the library is linked statically this routine needs to be called explicitly before any use of other routines (except a few ones like winx_print).

Parameters
[in]pebpointer to the Process Environment Block. May be NULL.
Returns
Zero for success, negative value otherwise.
Example 1 - a native application:
void __stdcall NtProcessStartup(PPEB Peb)
{
winx_init_library(Peb);
winx_kb_init();
// do something
winx_printf("Hello!");
winx_sleep(1000);
winx_exit(0);
}
Example 2 - a command line tool (zenwinx is linked statically):
int main(void)
{
winx_init_library(NULL);
// do something
printf("Hello!");
winx_sleep(1000);
winx_unload_library();
return 0;
}
Example 3 - a command line tool (zenwinx is linked dynamically):
int main(void)
{
// do something
printf("Hello!");
winx_sleep(1000);
return 0;
}
