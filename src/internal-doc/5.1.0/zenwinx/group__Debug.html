void winx_dbg_print ( char *  format,
  ... 
)

Delivers a message to the Debug View program and appends it to the log file as well.

Note
  • If : $LE appears at end of the format string, the last error code will be appended to the message as well as its description.
  • If : $NS appears at end of the format string, the NT status code of the last operation will be appended to the message as well as its description.
  • Not all system API set last status code. Use winx_dbg_print_ex to catch the status for sure.

char* winx_get_error_description ( unsigned long  status)

Retrieves a human readable explanation of the NT status code.

Parameters
[in]statusthe NT status code.
Returns
A pointer to string containing the status explanation.
Note
This function returns explanations only for well known codes. Otherwise it returns an empty string.
Example:
printf("%s\n",winx_get_error_description(STATUS_ACCESS_VIOLATION));
// prints "access violation"
