Inserts an item to double linked list.

Allocates memory for an item to be inserted.

Parameters
[in,out]pheadpointer to a variable pointing to the list head.
[in]prevpointer to an item preceeding to the new item. If this parameter is NULL, the new head will be inserted.
[in]sizethe size of an item to be inserted, in bytes.
Returns
Pointer to the inserted list item. NULL indicates failure.
