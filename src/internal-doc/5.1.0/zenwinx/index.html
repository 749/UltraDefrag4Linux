ZenWINX?

It is a framework for Windows Native applications development. Such applications are running during the Windows boot process in a manner similar to chkdsk. The following native applications are well known too:

  • xmnt2002.exe is a native part of the Partition Magic software
  • smss.exe is one of the most important Windows components participating in the Windows boot process

Using ZenWINX library you may easily create your own native application. This library gives you over 100 functions which highly simplifies the Windows native development. Also it contains a Native Development Kit header file with a lot of definitions required for the native development.

ZenWINX is fully compatible with Windows NT 4.0, Windows 2000, Windows XP, Windows Server 2003, Windows Vista, Windows Server 2008, Windows 7 and all of the 64-bit editions of Windows.

Projects

ZenWINX library is currently used in the following projects:

  • UltraDefrag - a little shell included there allows to defragment hard drives as well as shutdown and reboot the computer before the Windows logon.
  • Native shell - this little program runs before the Windows logon too. It allows to navigate through the file system, copy and delete files, display directory listings and more.

If you know other projects not listed here, please send me a note and I will expand the list.

Compilation

Actually at this moment ZenWINX is a part of the UltraDefrag project. You should therefore download the latest sources of them to compile ZenWINX library. Detailed information is here.
