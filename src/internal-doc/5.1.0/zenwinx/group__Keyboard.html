char* winx_get_error_description ( unsigned long  status)

Retrieves a human readable explanation of the NT status code.

Parameters
[in]statusthe NT status code.
Returns
A pointer to string containing the status explanation.
Note
This function returns explanations only for well known codes. Otherwise it returns an empty string.
Example:
printf("%s\n",winx_get_error_description(STATUS_ACCESS_VIOLATION));
// prints "access violation"
