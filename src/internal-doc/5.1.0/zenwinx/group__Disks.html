winx_volume_region* winx_get_free_volume_regions ( char  volume_letter,
int  flags,
volume_region_callback  cb,
void *  user_defined_data 
)

Retrieves list of free regions on the volume.

Parameters
[in]volume_letterthe volume letter.
[in]flagscombination of WINX_GVR_xxx flags.
[in]cbaddress of procedure to be called each time when the free region is found on the volume. If callback procedure returns nonzero value, the scan terminates immediately.
[in]user_defined_datapointer to data passed to the registered callback.
Returns
List of free regions, NULL indicates that either disk is full (unlikely) or some error occured.
Note
  • It is possible to scan disk partially by requesting the scan termination through the callback procedure.
  • Callback procedure should complete as quickly as possible to avoid slowdown of the scan.
